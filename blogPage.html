  • Question No 3:What is an event loop? How it work?

    • Answer :The event loop is the secret behind JavaScript’s asynchronous programming. The event loop is the term given to the process of the waiting for the queue to receive a message synchronously. The increment that the event loop moves in is called a ‘tick’, and every time it ‘ticks’ it checks if the call stack is empty, if so , it adds the top function in the event queue to the call stack and executes it. Once it is finished processing this function it starts ticking again.
